# 👾 8-Bit Galaxy V2.0

¡Bienvenido a la versión final de **8-Bit Galaxy**!

Este proyecto es el resultado del trabajo final para el curso de **Talento Tech**. Se implemento un apartado de "Merchandising" dinámico orientado a videojuegos retro, con un catálogo interactivo, carrito de compras funcional y diseño responsivo.

## ✨ Características Principales

* **Merchandising Dinámico:** Consumo de datos desde la [Fake Store API](https://fakestoreapi.com/) para mostrar productos de manera aleatoria.
* **Carrito de Compras:** Funcionalidad completa con `localStorage` (persistencia de datos), edición de cantidades, eliminación y cálculo de total.
* **Menú Hamburguesa:** Navegación optimizada para dispositivos móviles con transiciones suaves.
* **SEO y Accesibilidad:** Uso de etiquetas semánticas y buenas prácticas de optimización para motores de búsqueda.
* **Formulario de Contacto:** Integración funcional con [Formspree](https://formspree.io/).
